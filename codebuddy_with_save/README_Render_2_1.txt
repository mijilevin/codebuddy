CodeBuddy ל-Render - צ'אט 24/7
============================

שלבים:
1. צור חשבון ב-render.com (חינם)
2. צור API key ב-platform.openai.com
3. העלה את התיקיה הזאת ל-GitHub
4. ב-Render: New → Blueprint → חבר את ה-GitHub
5. בהגדרות Environment הוסף:
   OPENAI_API_KEY = ה-key שלך
6. Deploy - תקבל כתובת https

המפתח נשמר בצד שרת, לא חשוף למשתמשים.
